Spring MVC Mapping 테스트

1. 메소드에 요청 매핑하기

GET 방식 메뉴 등록 : /menu/regist

POST 방식 메뉴 등록 : /menu/regist

GET 방식 메뉴 수정 : /menu/modify

POST 방식 메뉴 수정 : /menu/modify

GET 방식 메뉴 삭제 : /menu/delete

POST 방식 메뉴 삭제 : /menu/delete




2. 클래스에 요청 매핑하기

GET 방식 메뉴 등록 : /order/regist

POST 방식 메뉴 수정 : /order/regist

POST 방식 메뉴 삭제 : /order/delete

GET 방식 주문 상세보기 : /order/detail/{orderNo}
